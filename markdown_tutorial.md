## Изображения

Для вставки картинки, если это локальный файл, то нужно указать его имя:

`![image](image.jpg)`

Аналогично с изображениями из интернета, только вместо имени файла, нужно прописать его адрес в сети:

`![image](https://internet-image.com)`
